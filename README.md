**BlogNest** is a full-stack blogging platform built with:

- **Node.js**
- **Express.js**
- **EJS (Embedded JavaScript Templating)**
- **MongoDB + Mongoose**
- **Tailwind CSS**
- **Multer** (for file uploads)
- **express-session** + **connect-mongo** (for user session management)
